【功能说明】

Web UI 包含以下主要功能块：

1. 问答界面
   ├─ 输入自然语言问题
   ├─ 显示 AI 回答
   ├─ 显示置信度、耗时、来源
   └─ 支持 Enter 快速提问

2. 知识库搜索
   ├─ 关键词搜索
   ├─ 相似度评分
   ├─ 文档分类显示
   └─ 支持 Enter 快速搜索

3. 服务状态监控
   ├─ 实时健康状态
   ├─ 响应时间显示
   ├─ 状态指示器（绿/橙/红）
   └─ 30 秒自动刷新

4. 系统信息展示
   ├─ 运行环境
   ├─ 版本号
   └─ 更新时间

5. Prompt 角色列表
   ├─ 销售顾问
   ├─ HR 顾问
   ├─ 技术顾问
   └─ 财务顾问

6. 企业系统展示
   ├─ ERP 系统
   ├─ HRM 系统
   ├─ CRM 系统
   └─ 财务系统

7. Agent 工具列表
   ├─ ERP 销售查询
   ├─ HRM 员工查询
   └─ 财务预算查询

【API 端点】

Web UI 后端提供以下 API 端点：

GET   /                                    返回主页 HTML
GET   /health                              健康检查
GET   /api/services/status                 获取所有服务状态
POST  /api/qa/ask                          发送问题
GET   /api/prompts                         获取 Prompt 列表
GET   /api/rag/documents                   获取知识库文档
POST  /api/rag/search                      搜索知识库
GET   /api/agent/tools                     获取 Agent 工具
GET   /api/integration/status              获取集成状态
GET   /api/integration/erp/sales/*         获取 ERP 销售数据
GET   /api/llm/models                      获取 LLM 模型列表
