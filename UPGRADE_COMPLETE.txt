🔄 维护和升级

修改 UI：
  1. 编辑 services/web_ui/static/index.html
  2. docker cp ... ai_lite_web_ui:/app/static/
  3. 刷新浏览器

修改后端：
  1. 编辑 services/web_ui/main.py
  2. docker restart ai_lite_web_ui

添加新功能：
  1. 在 HTML 中添加新页面
  2. 在 JavaScript 中添加导航逻辑
  3. 在后端添加新 API 端点
