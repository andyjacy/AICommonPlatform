## 🔧 技术亮点

### 前端
- **纯 HTML5/CSS3/JavaScript**：无框架依赖
- **响应式设计**：支持各种屏幕尺寸
- **800 行代码**：完整功能，代码简洁
- **平滑动画**：专业的视觉效果

### 后端
- **FastAPI**：高性能 Python 框架
- **Mock 数据生成**：无数据库依赖
- **错误自动回退**：确保用户体验
- **轻量级运行**：内存占用小

### 集成
- **真实 + Mock 混合**：实际服务时用真实数据，服务故障时用 Mock 数据
- **零迁移成本**：前端无需修改
- **渐进式升级**：先用 Mock，再替换为实际服务

## ❓ 常见问题

**Q: 这些数据是真实的吗？**
A: 前 7 个在线容器的数据是真实的，其他展示页面使用精心设计的 Mock 数据来演示能力。

**Q: 为什么问题的回答不同？**
A: 系统有内置的关键词匹配，不同的问题会返回不同领域的回答。

**Q: 能否连接真实的数据库？**
A: 可以。修改 FastAPI 后端，更换 Mock 数据生成器为真实数据查询即可。

**Q: UI 页面支持移动设备吗？**
A: 是的，使用了响应式设计，支持各种设备。
