启动和停止：
  docker-compose up -d         # 启动所有服务
  docker-compose down          # 停止所有服务
  docker-compose restart       # 重启服务

查看日志：
  docker-compose logs -f       # 查看全部日志
  docker-compose logs -f web_ui        # 查看 Web UI 日志
  docker-compose logs -f qa_entry      # 查看 QA 服务日志

查看状态：
  docker-compose ps            # 查看容器状态
  docker stats                 # 查看资源使用

测试：
  python3 scripts/test_api.py   # 运行 API 测试

清理：
  docker-compose down -v       # 停止并删除所有卷

❓ 问题诊断

问题：Web UI 无法打开
答案：
  1. 检查服务是否启动：docker-compose ps
  2. 查看日志：docker-compose logs -f web_ui
  3. 检查端口：netstat -an | grep 3000

问题：提问超时
答案：
  1. 检查所有服务状态：Web UI 右侧面板
  2. 查看 QA Service 日志
  3. 检查数据库连接

问题：知识库无结果
答案：
  1. 确保 RAG Service 正常运行
  2. 检查是否有文档上传
  3. 修改搜索关键词
