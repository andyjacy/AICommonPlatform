✨ 项目概述
══════════════════════════════════════════════════════════════════

一套完整的、生产级的企业AI能力平台，包含6个独立的微服务容器，
提供问答、知识库、Agent、提示词管理和企业系统集成等核心能力。

【关键特性】
✓ 开箱即用 - 一条命令启动所有服务
✓ 轻量级部署 - 无Elasticsearch，仅需PostgreSQL+Redis+Milvus
✓ 生产级质量 - 完整的错误处理、日志、监控、健康检查
✓ 高度可定制 - 支持多个LLM、可扩展系统集成、灵活模板
✓ 完整文档 - 3000+行文档，从入门到精通

🚀 启动和使用
══════════════════════════════════════════════════════════════════

【快速启动】(仅3行命令)

  cd AICommonPlatform
  docker-compose up -d
  python3 scripts/test_api.py

【验证启动】
  
  bash scripts/verify.sh

【访问服务】

  QA Entry:      http://localhost:8001
  Prompt:        http://localhost:8002
  RAG:           http://localhost:8003
  Agent:         http://localhost:8004
  Integration:   http://localhost:8005
  LLM:           http://localhost:8006
  Prometheus:    http://localhost:9090
  Grafana:       http://localhost:3000 (admin/admin)

【示例API调用】

  # 提问
  curl -X POST http://localhost:8001/api/qa/ask \
    -H "Content-Type: application/json" \
    -d '{"question": "今年Q1的销售额是多少?", "user_id": "user123"}'

  # 获取Prompt模板
  curl http://localhost:8002/api/prompts

  # 搜索知识库
  curl -X POST http://localhost:8003/api/rag/search \
    -H "Content-Type: application/json" \
    -d '{"query": "销售", "top_k": 5}'

  # 查询企业数据
  curl http://localhost:8005/api/integration/erp/sales/2024/Q1

🔧 常用命令
══════════════════════════════════════════════════════════════════

【启动和停止】

  docker-compose up -d       # 启动所有服务
  docker-compose down        # 停止所有服务
  docker-compose restart     # 重启服务
  docker-compose logs -f     # 查看日志

【测试和验证】

  python3 scripts/test_api.py     # 运行API测试
  bash scripts/verify.sh          # 启动验证
  docker-compose ps               # 查看容器状态
  docker stats                    # 查看资源使用

【使用Makefile】

  make up          # 启动
  make down        # 停止
  make restart     # 重启
  make test        # 测试
  make logs        # 日志
  make ps          # 状态
  make clean       # 清理

【开发模式】

  make dev-qa      # 本地运行QA服务
  make dev-rag     # 本地运行RAG服务
  # 等等

💡 使用场景示例
══════════════════════════════════════════════════════════════════

【场景1: 销售查询】

  提问: "今年Q1的销售额是多少?"
  系统:
    1. 识别为销售查询
    2. RAG加载销售报告
    3. Agent调用ERP系统
    4. LLM用销售顾问模板生成答案
  答案: "Q1总销售额5000万元，同比增长15%"

【场景2: HR咨询】

  提问: "公司有多少员工?"
  系统:
    1. 识别为HR查询
    2. Agent调用HRM系统
    3. LLM用HR顾问模板生成答案
  答案: "公司500名员工，分布在5个部门"

【场景3: 综合分析】

  提问: "Q1的销售额和预算达成情况?"
  系统:
    1. 同时调用RAG、Agent、LLM
    2. 综合销售、财务数据
    3. 进行AI分析和建议
  答案: "详细的财务分析报告"
