## 📞 需要帮助？

### 常见问题

**Q: 无法访问？**
A: 检查地址是否为 `http://localhost:3000`（不是 https）

**Q: 页面很慢？**
A: 刷新页面，或检查浏览器是否有很多标签页打开

**Q: 搜索没有结果？**
A: 系统会自动生成相关的示例结果

**Q: 某个模块没有内容？**
A: 这是正常的，Mock 数据会自动显示

### 获取帮助

1. **查看浏览器控制台** - F12 打开开发者工具
2. **查看 Docker 日志** - `docker logs ai_lite_web_ui`
3. **查看相关文档** - ENHANCED_UI_GUIDE.md
4. **重启服务** - `docker restart ai_lite_web_ui`
